# Week4 项目

本目录包含第4周的学习内容。

## 目录结构

- `week4/` — 第4周作业与练习
  - `0608.ipynb` / `0608.py` — 成绩可视化分析 + IoU 计算 + 特征相似度检索
  - `0609.ipynb` — 后续练习
  - `0610.ipynb` — 图像处理综合练习
  - `figures/` — 图片资源
  - `file/` — 数据文件（特征库）
    - `feat.txt` — 特征库数据
    - `target_feat.txt` — 目标特征数据
  - `images/` — 原始图片素材
  - `images_resized/` — 批量缩放后的图片
  - `data/` — 小黄人贴图数据集
    - `labels.txt` — 标注坐标
    - `xhr_imgs/` — 小黄人图片
    - `bg_imgs/` — 背景图片

## 内容概要

1. **成绩可视化分析** — 使用 Matplotlib 绘制柱状图、直方图、散点图、折线图
2. **IoU 计算** — 目标检测中候选框与真实框的交并比计算
3. **特征相似度检索** — 基于余弦相似度查找最相似的特征
4.**图像处理基础（0610.ipynb）**
   - 创建 RGB / 黄绿蓝对角渐变图
   - 图片切片、通道分离与合并
   - 批量缩放图片
   - 九宫格拆分与拼接还原
   - 添加红色边框
   - 随机/按标注坐标贴图（小黄人贴到背景图）
