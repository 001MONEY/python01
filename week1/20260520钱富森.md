## 2.3交换两个变量的值 交换前 a = 1， b = 2 交换后 a = 2, b = 1把你能想到的实现方式写出来

```python

b = 2
print(f"交换前:a = {a}, b = {b}")
a, b = b, a
print(f"交换后:a = {a}, b = {b}")
```

交换后:a = 2, b = 1
